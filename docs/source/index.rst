Welcome to Laia's documentation!
===================================

**Laia's** documentation pretends to make peoples life that work in this project easier.
It pulls the `Open Food Facts database <https://world.openfoodfacts.org/>`_ and and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

Check out :doc:`workingwithhow2sign` section for that step documentation.

.. note::

   This project is under active development.
